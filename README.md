# Erics118 Tap

## How do I install these formulae?

`brew install erics118/tap/<formula>`

Or `brew tap erics118/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "erics118/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
